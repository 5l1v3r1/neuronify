Neuronify QML Types

This is a list of all documented QML Types in Neuronify.

ACClamp

An alternating current generator which can suply input to neurons

CurrentClamp

A direct current generator which can suply input to neurons

FileManager

A type to contain the save/load features

Neuronify

This type holds the application

PoissonGenerator

An spike generator which can suply input spikes to neurons

Retina

A visual sensor that can be connected to neurons to generate activity based on the receptive field of the sensor and the visual stimuli captured by the camera

Speaker

A speaker that emits a sound whenever a connected neuron fires

Voltmeter

A voltmeter that neurons can connect to in order to show their voltage trace

VoltmeterControls

Contains the user controls for the voltmeter
